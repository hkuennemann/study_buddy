## Setting up the environment
1. Create (or reuse) a virtual environment:
```bash
python3 -m venv name_of_venv
```
2. Activate it:
```bash
source name_of_venv/bin/activate
```
3. Install the dependencies:
```bash
pip install -r requirements.txt
```
